## 猜數字遊戲

這是一個簡單的猜數字遊戲程式。

### 如何運行
1. 確保你有Python環境。
2. 運行 `python guess_number.py`。
3. 跟隨提示輸入你的猜測。

遊戲規則：
- 程式會隨機選擇一個1到100的數字。
- 你需要猜測這個數字。
- 程式會告訴你猜測是太小還是太大。
- 直到猜對為止。

### 範例輸出
```
歡迎來到猜數字遊戲！
我已經想好一個1到100的數字，請猜猜看。
請輸入你的猜測：50
太小了！再試試。
請輸入你的猜測：75
太大了！再試試。
請輸入你的猜測：60
恭喜你猜對了！答案是 60。
你總共猜了 3 次。
遊戲結束。
```
